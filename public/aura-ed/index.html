Inclusive AI for UAE classrooms

A bilingual learning platform that closes the gap between national tech ambition and day-to-day learning reality.

Aura-Ed combines adaptive scaffolding, SEND-aware design, low-bandwidth resilience, and regional compliance into one product foundation for schools across the UAE.

Run adaptive simulation View technical README

System posture

Designed for UAE hosting, Arabic-first access, device equity, and teacher-visible interventions.

Privacy layer

Edge AI keeps sensitive attention signals in the browser.

Only engagement scores travel upstream. Raw video, audio, and biometric traces stay local unless the institution explicitly enables them.

Architecture

A decentralized microservices model that treats access, inclusion, and compliance as core infrastructure.

Each domain is isolated so schools can scale services independently while respecting local residency, accessibility, and interoperability needs.

Recommended scaffold actions

    Feature set

    The first release is shaped around inclusion, teacher readiness, and local relevance.

    These modules map directly to the ground-level issues highlighted across UAE online learning research.

    Compliance and safety

    Every major workflow is framed by privacy, interoperability, accessibility, and integrity design.

    The goal is to make compliance a product feature instead of a last-minute paperwork exercise.

    Implementation roadmap

    A phased build that starts lean, proves value quickly, and expands responsibly.

    Each phase keeps scope tight while preparing the platform for institutional rollout.
